## [OpenAI Codex 新增导入其他Agent配置功能和宠物功能](https://chatgpt.com/codex/switch-to-codex/) `#1`
> **OpenAI**推出了`Switch to Codex`迁移功能，支持用户一键将其他Agent的设置、插件、技能和历史对话等工作流迁移至`Codex`。
>
> 同时，`Codex`还上线了宠物功能。用户输入`/pet`命令即可领养默认宠物，或通过`/hatch-pet`技能创建自定义宠物来掌握状态信息。

**OpenAI** 近期推出了一系列 **Codex** 新功能，旨在降低其他编码 Agent 用户的迁移门槛并提升日常使用体验。

其中，官方正式上线了“导入设置”功能，支持用户通过官方应用在几步之内从其他 Agent 导入设置、插件、`Agent`、项目配置等，从而在工作流切换时减少中断。

与此同时，`Codex` 还新增了名为"Pets"的趣味功能。用户可通过 `/pet` 命令唤醒宠物，并支持通过安装 `hatch-pet` 技能进行自定义创建。

## [xAI API平台上线自定义声音功能](https://x.ai/news/grok-custom-voices) `#3`
> **xAI**宣布其 **API** 正式上线自定义声音与语音库功能。
>
> 用户只需提供约 **一分钟** 自然语音，即可在 **两分钟** 内完成验证并快速克隆专属声音。
>
> 同时还能选用超 **80** 种覆盖 **28** 种语言的预设声音。

xAI 通过官方渠道宣布，其 API 正式上线**“自定义声音”**与**“语音库”**两项新功能。

在**自定义声音**方面，用户只需提供约**一分钟**的自然语音录音，并在控制台完成验证短语与声纹嵌入的双重验证，即可在不到**两分钟**内生成专属声音模型。

该功能支持应用于`Grok 文本转语音接口`和`语音 Agent API`，且不产生额外费用。

同时上线的**语音库**支持统一管理，内置声音已扩展至超过**80 种**，覆盖**28 种语言**。

xAI 强调，因需实时验证与声纹比对，无法从预先存在的录音中克隆他人声音，从而防止滥用。

## [Vercel Labs 推出 AI CLI 支持终端统一生成各类媒体](https://github.com/vercel-labs/ai-cli) `#4`
> **Vercel Labs** 推出了开源的 AI 命令行工具，让用户能直接在终端里通过简单的命令生成文本、图像和视频。
>
> 该工具支持通过管道操作，串联生成步骤、进行多模型对比以及无需原生依赖的终端内联预览。

**Vercel Labs** 近日开源 **AI CLI** 工具，支持用户在终端通过命令直接生成文本、图像和视频。该工具基于 **AI SDK** 和 `Gateway` 构建，可统一接入数百种模型。

其核心亮点包括：一是支持管道操作，可将不同命令的输入输出直接串联；二是具备内联预览功能，若终端支持 `Kitty` 协议，无需原生依赖即可由 `WebAssembly` 解码显示图像及视频关键帧。

此外，该工具支持多模型生成与对比，设定文本和图像命令超时 `120` 秒、视频命令 `300` 秒。用户需提供 `AI Gateway` 或特定提供商的 **API** 密钥方可使用。

## [Meta AI 推出 Autodata，Agent 自主构建高质量训练数据](https://facebookresearch.github.io/RAM/blogs/autodata/) `#7`
> **Meta AI**的 **RAM** 团队推出了 `Autodata` 框架，该框架通过一个主 Agent 协调四个子 Agent 来严格筛选问答对。
>
> 这一机制让 AI Agent 能像数据科学家一样自主迭代，并生成高质量的训练数据。

**Meta AI** 的 **RAM** 团队推出 `Autodata` 框架，使 `AI Agent` 能自主构建高质量的训练与评测数据。该方案由一个主 `Agent` 协调四个子 `Agent`，依严格条件迭代筛选样本。

实验中，该框架处理超 **1 万**篇论文，产出 **2117**个有效 `QA` 对。它将强弱求解器得分差距从传统方法的 **1.9**个百分点扩至 **34**个百分点。用此数据训练的 `Qwen-3.5-4B` 模型表现显著优于基线。其元优化机制将数据生成验证通过率从 **12.8%**提升至 **42.4%**。

团队计划在 arXiv 发布完整技术报告。

美国国会加速推进禁止面向未成年人提供`AI`聊天机器人的立法。

参议院司法委员会**一致通过**由两党议员**Josh Hawley**和**Richard Blumenthal**提出的《**GUARD 法案**》。该法案要求`AI`公司实施年龄验证，禁止向未成年人提供`AI`伴侣，并强制披露非人类身份。

众议院也提出配套法案。对诱导未成年人性行为或鼓励自杀的公司将施加刑事处罚。

>
