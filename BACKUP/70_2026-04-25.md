## [DeepSeek 发布 DeepSeek-V4 系列模型](https://api-docs.deepseek.com/zh-cn/news/news260424) `#1`
> **DeepSeek**正式发布并开源了全新的`DeepSeek-V4`系列预览版模型，包含 Pro 和 Flash 两个版本，均支持 `100万` 超长上下文。
>
> 官方称，`DeepSeek-V4-Pro` 性能比肩顶级闭源模型，Agent 能力大幅提升，拥有丰富的世界知识和世界顶级的推理性能。
>
> 而 `DeepSeek-V4-Flash` 则是更快捷高效的经济之选。两款模型已在官网、App 以及官方 API 中上线。

**DeepSeek** 近日正式发布并开源了全新的 `DeepSeek-V4` 系列预览版模型。该系列包含参数量达 **1.6T**（激活 **49B**）的 `DeepSeek-V4-Pro`，以及参数量 **284B**（激活 **13B**）的 `DeepSeek-V4-Flash` 两款 `MoE` 架构模型。

该系列模型全面支持 **1M** 超长上下文长度和最大 **384K** 的输出长度。在架构上，其首创了结合 `CSA` 与 `HCA` 的混合注意力机制以及 `mHC` 技术。官方数据显示，相较于前代模型，其在推理和长上下文效率上大幅提升。

目前，用户可通过官网、App 及 API 体验该系列模型，其 API 兼容 `OpenAI` 和 `Anthropic` 格式。官方称 `DeepSeek-V4-Pro-Max` 模式在 `Agentic Coding` 等基准测试中取得顶级表现，稳居当今最佳开源模型之位。

此外，API 服务提供了思考与非思考模式及不同强度的切换功能。原有的 `deepseek-chat` 与 `deepseek-reasoner` 模型名将在**三个月后**停止使用。

## [阶跃星辰发布 StepAudio 2.5 ASR 语音识别模型](https://mp.weixin.qq.com/s/jGeJxEsGHaLNT00BoqcTZw) `#3`
> **阶跃星辰**发布了语音识别模型 `StepAudio 2.5 ASR`。官方称该模型能一次性端到端完整处理 **30 分钟** 的音频，中英双语识别精度均达到业内顶尖水平。
>
> 该模型已上线 **Step Plan**，并开放了体验中心及相关 Demo 供用户体验。

**阶跃星辰**发布了自动语音识别模型 `StepAudio 2.5 ASR`。该模型基于 `ASR+MTP-5` 深度融合架构，将大语言模型（`LLM`）的推理加速技术引入语音识别领域，实现了速度与精度的双重提升。

官方数据显示，`StepAudio 2.5 ASR` 的解码速度较前代提升 **4 倍**，时延降低 **60%**，推理峰值达 **500 tokens/s**，推理成本直降 **80%**。

该模型复用了 `LLM` 原生的 **32K** 上下文窗口，单次可端到端完整处理长达 **30 分钟** 的连续音频，解决了传统切片拼接方案导致的上下文断裂问题。同时，官方宣称其在多项中英文权威开源测试集上达到业内 **SOTA** 精度水平。

目前，`StepAudio 2.5 ASR` 已全量上线**阶跃星辰开放平台**和 **Step Plan**，并开放了体验中心及相关 Demo 供用户使用。

## [Anthropic 发布名为 Project Deal 的实验研究结果](https://www.anthropic.com/features/project-deal) `#8`
> **Anthropic**开展了一项名为`Project Deal`的实验，让 **Claude**代替员工在虚拟市场中自主进行物品交易和讨价还价。
>
> 实验发现，AI 能独立完成上百笔真实交易。使用高级模型 `Opus`的 AI 在成交价和成交量上占据明显优势。

**Anthropic**近日发布了名为 **Project Deal** 的实验研究结果，该实验于**2025 年 12 月**在其旧金山办公室开展，旨在探索 `AI Agent` 代理人类进行商业交易的可行性及市场影响。

实验中，**69 名员工**每人获得**100 美元**预算，由 **Claude** 通过 `intake interview` 了解其买卖意向、期望价格与谈判风格后生成定制 `Agent`，随后在公司 **Slack** 上的四个并行市场中自主完成交易。其中仅 **Run A** 为基于 `Claude Opus 4.5` 的真实运行，员工将据此实际交换商品。**Run D** 为全 `Opus` 模拟运行，**Run B** 与 **Run C** 则随机混合使用 `Claude Opus 4.5` 与 `Claude Haiku 4.5`，且全程无人工干预。

最终真实运行针对超过**500 件**上架商品达成**186 笔**交易，总价值超过**4000 美元**。官方数据显示，在混合模型的模拟市场中，`Opus Agent` 客观上显著优于 `Haiku`。前者作为卖家能为同一商品多争取约**2.68 美元**，作为买家则能少支付约**2.45 美元**，且平均多完成约**2 笔**交易。然而参与者在事后的满意度与公平性感知上并未体现出显著差异，公平度评分几乎相同。

实验还发现，用户要求的攻击性或礼貌性谈判风格等自定义指令对成交结果无统计显著影响，但模型质量差异带来的议价能力分化已十分明显。值得注意的是，**46%**的参与者表示未来愿意为此类服务付费。

但 **Anthropic** 也警示，`Agent` 质量差距可能在真实市场中悄然加剧不平等，并带来 `jailbreaking`、`prompt injection` 及法律政策滞后等风险。
