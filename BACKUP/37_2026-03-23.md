## [微信正式推出ClawBot插件，支持接入OpenClaw](https://mp.weixin.qq.com/s/o_FPvJ0tY6aGqGn4Ea7Rpw) `#1`
> 微信发布了官方龙虾插件 **ClawBot**，用户将微信更新至最新版并在设置中启用该插件后，即可在终端执行安装指令扫码绑定。
>
> 目前该功能仅限个人单聊，不支持群聊及自动化操作。
>
> 腾讯云 **Lighthouse**、**WorkBuddy** 及 **QClaw** 等产品也已同步适配微信通道。

微信发布了 **ClawBot** 插件，支持将 **OpenClaw** 接入微信，使用户能够在微信聊天界面直接调用该AI助手处理任务。

用户需将微信更新至最新版本，在“我→设置→插件”中启用 **ClawBot**，并执行终端指令 `npx -y @tencent-weixin/openclaw-weixin-cli@latest install` 扫码绑定账号。

目前该插件正在逐步放量，仅限个人单聊，不支持加入微信群或自动化操作微信。

与此同时，腾讯云 **Lighthouse**、自研工具 **WorkBuddy** 及 **QClaw** 等产品也已完成适配，用户可通过各产品客户端的“微信通道”扫码直连。

## MiniMax 调整 Token Plan 额度方案，提供多模态模型每日配额 `#4`
> **MiniMax** 调整了 Token Plan 详细方案，文本模型依旧按请求次数计算消耗，保持每 **5小时** 滚动重置，多模态模型则按套餐类型提供不同的每日固定配额。

**MiniMax** 调整了其 Token Plan 的详细用量额度方案，该方案针对不同模型实施了差异化的重置机制与分级配额。

Token Plan 的额度计算依据模型类型分为两类：**M2.7** 及其高速版本按请求次数计，每 **5 小时** 滚动重置；而语音、视频、音乐及图像等其他模型则按日配额计算，每日重置。
